The Map Method

Our friendly enumberable helper, the map method

August 2, 2015

One of the basic tenets of coding is DRY â€” don't repeat yourself. Still, one of the major functions of coding is to manipulate large quantities of data, often running operations on lots of them at the same time.The map method runs on an object, such as an array, hash, or range, and executes your given code on each element of the method's receiver. The output is a new array (or, if you didn't put in a code block, just an enumerator).

Let's look at an example to help us understand this better. Let's say we have an array that contains a list of names. We want to capitalize each of the first letters of those names. This is what our code would look like:

["sammy", "alex", "jordan", "addison", "jamie", "taylor"].map {|name| name.capitalize} => ["Sammy", "Alex", "Jordan", "Addison", "Jamie", "Taylor"]

A few pointers: This actually works the same as the collect method; map is just more commonly used. If collect makes more sense to you, as it does to me, it might be helpful to use that for iterating through enumerators. You can also use the method on hashes, but remember to pass in two variable inputs to represent the keys and values.
