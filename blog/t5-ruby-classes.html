Pugs as Classes

Learning about Ruby classes with pugs

August 9, 2015

Given that Ruby is an object-oriented language, it's important to understand how they relate to classes. Objects are the building blocks of everything we do in Ruby. We make them, we ask them for information, we give them information, and we generally fill them up with data to use. When we have similar objects that we want to have them same abilities and set of attributes, we can cluster them into a class and define them with a class blueprint. That way, we can make objects out of their class blueprints and have them come with a template of information.

This is maybe confusing, so let's call in some friendly pugs to help us break it down.

We start by defining it with its name, which is capitalized. This is important because Ruby expects classes to start with a capital letter. The first method we call initializes it and should include everything we need to start, such as arguments that need to be passed to each new instance of the class. If we want to use those arguments later on, which we surely will, we should save them as instance variables. Instance variables are similar in concept to local variables that we've been using elsewhere, but they're preceeded by an @ and are specific to the class you're in. That means even if I define an instance variable in my intialize method, I can call it in another method I define within the class.

This is a simple class definition for a pug:

class Pug
  def initialize(name, age)
    @name = name
    @age = age
  end

  def bark
    puts "Woof woof"
  end

  def introduce
    puts "Hi, my name is #{name} and I am #{age} years old."
  end
end

Now that we have a general template for pugs, we can create specific ones, such as Emmy, Charlie, and B.

emmy = pug.new("Emmy", 13)
charlie = pug.new("Charlie", 4)
b = pug.new("B", 7)
johnny = pug.new("Johnny", 1)

Each of these dogs has some built-in methods. If we call

emmy.bark

the output is

"Woof woof"

and if we ask Charlie to introduce herself with

charlie.introduce

we get

"Hi, my name is Charlie and I am 4 years old."

Emmy, Charlie, and B all hope that helps you understand classes better!
